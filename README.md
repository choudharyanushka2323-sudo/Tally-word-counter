here# Tally — Word & Character Counter

A clean, instant word and character counter. No sign-up, no ads — just type or paste your text and see the count update live.

## Why I built this

I've used tools like this myself when writing tweets, bios, and captions that need to fit under a strict character limit. Most counters online are cluttered with ads or hide the actual count below the fold. This one puts the number front and center and updates as you type.

## Features

- Live word and character count, switchable with one click
- Sentence count and estimated reading time
- Optional character/word limit with a visual warning when you go over
- Copy-to-clipboard and clear buttons
- Fully responsive, keyboard accessible, respects reduced-motion preferences

## Tech stack

Plain HTML, CSS, and vanilla JavaScript — no framework, no build step. Deploys directly on Vercel's free Hobby plan.
